# KannaAI — Product Brief

**Domain:** kannaai.com  
**Tagline:** Your personal cannabis concierge  
**Core Insight:** Weedmaps is a map with a search bar. KannaAI is a conversation that understands you.

---

## 1. The Two-Sided Platform

### Consumer App (Kanna Chat)
The primary experience is a conversational AI — not a search bar, not a map. The user talks to **Kanna**, their personal cannabis concierge.

### Business Portal (Kanna for Business)
Dispensaries get a dashboard to manage inventory, promotions, and visibility on the platform.

## 3. Consumer Features (Full List)

### Core
- **AI Chat Interface** — conversational recommendation engine (the main screen)
- **Strain Matching** — mood/symptom/effect-based recommendations
- **Dispensary Search** — radius-based, filtered by stock availability
- **Live Navigation** — integrated maps with turn-by-turn directions
- **Strain Profiles** — detailed cards (THC/CBD%, effects, terpenes, reviews)

### Personalization
- **User Profile & Preferences** — tolerance level, preferred types, medical conditions
- **History & Favorites** — past recommendations, saved strains, visited dispensaries
- **Kanna Memory** — AI remembers past preferences and conversations
- **Personalized Feed** — deals and strains tailored to user profile

### Social & Trust
- **Strain Reviews** — user ratings with effect-based dimensions
- **Dispensary Reviews** — service quality, pricing, atmosphere
- **Community Picks** — trending strains in your area

### Commerce
- **Price Comparison** — same strain across nearby dispensaries
- **Deal Alerts** — notifications when favorite strains go on sale
- **Order Ahead** — reserve for pickup (where legally permitted)
- **Loyalty Points** — earn points for reviews, visits, referrals

### Business Features

#### Inventory Management
- **Menu Upload** — CSV/POS integration for bulk inventory sync
- **Real-Time Stock** — live availability reflected in Kanna recommendations
- **Strain Library** — pull from master database or add custom entries
- **Price Management** — set pricing, compare to local market rates

#### Promotions & Marketing
- **Deal Creator** — % off, BOGO, bundle deals, first-time visitor discounts
- **Featured Placement** — pay to boost visibility in Kanna recommendations
- **Push Notifications** — send deals to users who favorited your dispensary
- **Event Promotion** — list in-store events (420 celebrations, vendor days)

#### Analytics
- **Recommendation Tracking** — how often Kanna recommends your strains
- **Conversion Funnel** — views → "Take Me There" taps → estimated visits
- **Customer Demographics** — anonymized preference data
- **Competitive Insights** — how your pricing/selection compares locally

#### Compliance
- **License Verification** — verified badge for licensed dispensaries
- **Age Gate** — 21+ verification on consumer side
- **State Compliance** — adapts to local regulations per state

### Key Technology Choices

| Layer | Technology | Why |
|---|---|---|
| **Consumer App** | React Native (Expo) | Single codebase for iOS + Android, fast iteration |
| **Business Portal** | Next.js | SEO for dispensary pages, SSR, great DX |
| **API** | Node.js + tRPC or GraphQL | Type-safe, real-time capable |
| **AI Engine** | Claude API | Conversational recommendation, context memory |
| **Database** | PostgreSQL + Prisma | Relational data, strong typing |
| **Vector Search** | Pinecone or pgvector | Strain similarity matching by effects |
| **Maps** | Google Maps / Mapbox | Navigation, dispensary location |
| **Auth** | Clerk or Auth0 | Social login, age verification flow |
| **Hosting** | Vercel (web) + AWS/GCP (API) | Scalable, edge-optimized |
| **Push Notifications** | Expo Notifications + OneSignal | Deal alerts, order updates |

## 7. What KannaAI Does Better Than Weedmaps

| Weedmaps Pain Point | KannaAI Solution |
|---|---|
| Search-first UX — user must know what they want | **Conversation-first** — Kanna understands your mood and recommends |
| Filters reset every session | **Kanna remembers** your preferences across sessions |
| Price discrepancies between app and in-store | **Real-time POS sync** requirement for listed dispensaries |
| Negative reviews allegedly removed | **Transparent review system** with public moderation log |
| No real personalization | **Deep personalization** — Kanna learns your tolerance, preferences, medical needs |
| Navigation is an afterthought | **"Take Me There" is a first-class feature** with in-app navigation |
| Order processing unreliable | **Order-ahead as optional premium** — only enabled for dispensaries with proven fulfillment |

## 8. MVP Scope (Phase 1)

Ship the smallest thing that demonstrates the core differentiator:

1. **Kanna Chat** — AI conversation that recommends strains based on mood/effects
2. **Strain Database** — curated set of ~200 popular strains with effects/terpene data
3. **Dispensary Search** — location-based search with basic inventory (manual upload)
4. **"Take Me There"** — deep link to Google Maps / Apple Maps for navigation
5. **User Auth** — sign up, login, age verification (21+)
6. **Basic Business Portal** — dispensary signup, manual inventory management, basic profile

**Not in MVP:** Order-ahead, loyalty points, push notifications, POS integration, analytics dashboard, promoted placements.

---

## 9. Design Principles (Informed by Weedmaps Analysis)

1. **Conversation over chrome** — minimize UI, maximize the chat experience
2. **Green + clean** — cannabis branding without the stoner clichés. Modern, trustworthy, medical-grade feel
3. **Cards for content** — strain cards, dispensary cards, deal cards — consistent, tappable, informative
4. **Progressive disclosure** — don't overwhelm. Show what's needed, reveal detail on tap
5. **Speed to value** — user should get a recommendation within 2 messages
6. **Trust signals** — verified badges, transparent pricing, real reviews

## 10. Compliance Considerations

- **Age verification** — mandatory 21+ gate at signup
- **State-by-state legality** — geo-fence features to legal states only
- **Medical disclaimers** — Kanna is not a medical professional
- **No direct sales** — KannaAI is a recommendation/discovery platform, not a retailer
- **Data privacy** — health-related preferences require careful HIPAA-adjacent handling
- **Advertising regulations** — cannabis advertising varies by state
